## Get Started

1. Install the [`OrchardCoreContrib.Gdpr`](https://www.nuget.org/packages/OrchardCoreContrib.Gdpr/) NuGet package to your Orchard Core host project.
2. Go to the admin site
3. Select **Configuration -> Features** menu.
4. Enable the `GDPR` feature.
5. Go to **Configuration > Settings > GDPR** to configure the GDPR settings, if you want to customize the message and privacy content.
